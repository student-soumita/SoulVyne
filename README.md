# 💖 SoulVyn – Cinematic Love Experience

SoulVyn is a romantic interactive web application that combines love compatibility calculation, zodiac matching, cinematic animations, emotional storytelling, and relationship mini-games into one immersive experience.

Built using HTML, CSS, JavaScript, and a Python Flask backend.

---

# ✨ Features

- 💘 Love Compatibility Calculator
- ♈ Zodiac Match Analysis
- 🌌 Cinematic Intro Experience
- 💞 Interactive Heart Merge Game
- 🌸 Marriage Garden Popup System
- 💌 Shareable Love & Marriage Cards
- 🎵 Dynamic Romantic Music System
- 🌠 Animated UI & Particle Effects
- ❤️ Floating Hearts & Emotional Visuals
- 🔐 Local Storage Based Login System
- ⚡ Real-Time Backend Processing with Flask

---

# 🎮 Mini Games

## 💖 Heart Merge Game
Players must drag two hearts together before the timer ends to restore the emotional connection.

## 🌸 Marriage Garden
A romantic compatibility card system with dynamic emotional descriptions and share functionality.
